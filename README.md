# Claude Projects

A collection of small web apps built with Claude Code.

## Projects

### 📅 Calendar

A static month-view calendar app — no frameworks, no build step. Open `calendar/index.html` directly in any browser.

**Features**
- Month grid with prev/next navigation and a "Today" button
- Add, edit, and delete events via a modal form
- Events saved to `localStorage` (persists across page refreshes)
- US federal holidays pre-loaded for any year
- Sidebar with checkboxes to show/hide your events and holidays
- Responsive layout (mobile-friendly)
- Dark theme matching Google Calendar

**Usage**
1. Clone the repo
2. Open `calendar/index.html` in your browser
3. Click any day cell or the **+ New event** button to add an event

## Tech

Plain HTML, CSS, and JavaScript — no dependencies.
